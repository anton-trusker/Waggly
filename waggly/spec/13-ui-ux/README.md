|-------|-------|-------|
| Primary | #FF6B35 | CTAs, highlights |
| Secondary | #2EC4B6 | Success, health |
| Background | #FFFFFF / #1A1A1A | Light/dark |
| Text | #1A1A1A / #F5F5F5 | Content |
| Border | #E5E5E5 / #333333 | Dividers |
| Error | #FF4444 | Errors, alerts |
| Warning | #FFAA00 | Warnings |
| Success | #22CC88 | Confirmations |
