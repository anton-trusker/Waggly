## Sharing Methods

| Method | Use Case | Duration |
|--------|----------|----------|
| Co-owner Invite | Family members, partners | Permanent |
| Quick Share Link | Vets, temporary access | 24h default |
| QR Code Scan | Emergency, pet sitters | Configurable |
| Custom Link | Specific permissions | Configurable |

## Permission Presets

| Template | Purpose | Permissions |
|----------|---------|-------------|
| Full Access | Partners, family | View + Edit all |
| Vet Emergency | Veterinary care | View health + allergies |
| Pet Sitter | Caregivers | Care info + notes |
| Minimal | Curious relatives | Basic profile only |
| Custom | User-defined | Configurable |

---

## Key Features

- **Granular Permissions**: Control exactly what's shared
- **Expiring Links**: Time-limited access
- **Password Protection**: Optional link passwords
- **Access Tracking**: See who accessed and when
- **Easy Revocation**: Instantly revoke access
