## Overview

Waggly is an **AI-first digital pet passport platform** that revolutionizes pet health management through conversational AI, eliminating manual data entry and providing comprehensive health tracking for the €121B European pet care market.

## Quick Stats

- **Target Market**: €121B European pet care (91M households, 340M pets)
- **Platform**: React Native + Expo + Supabase
- **Revenue Model**: €4.99-€9.99/month subscriptions + affiliates
- **AI Capabilities**: OCR, conversational assistant, predictive health

## Document Standards

All PRD documents follow a consistent structure:
- **Overview** - Purpose and scope
- **User Stories** - As a [role], I want [feature], so that [benefit]
- **Requirements** - Functional and non-functional
- **Technical Spec** - API, database, integrations
- **UI/UX** - Component specifications
- **Success Metrics** - KPIs and measurement
