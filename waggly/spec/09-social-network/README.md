## Feature Summary

| Feature | Description | Status |
|---------|-------------|--------|
| Pet Profiles | Pet-centric social identity | Phase 3 |
| Posts | Photo, video, text, polls, check-ins | Phase 3 |
| Reactions | 6 reaction types (love, paws, etc.) | Phase 3 |
| Comments | Nested replies, mentions | Phase 3 |
| Following | Pet-to-pet and user follows | Phase 3 |
| Stories | 24-hour ephemeral content | Phase 3 |
| Groups | Public, private, secret communities | Phase 3.5 |
| Events | Local meetups and playdates | Phase 4 |
