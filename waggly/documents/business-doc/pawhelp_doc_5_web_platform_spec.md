### Web Vision & Design Principles
- Accessible, trustworthy, and informative user experience.
- Full feature parity with mobile application.
- Responsive design for desktop, tablet, and mobile browsers.
- Community-focused interface promoting action and transparency.
- Seamless onboarding for all user roles.

### Technical Stack
- React.js with Next.js for server-side rendering.
- TypeScript for robust, maintainable code.
- Styled Components for consistent UI styling.
- Redux for state management.
- Axios for API communication.
- React Hook Form & Yup for forms.
- React Query for data fetching & caching.

---

### Full User Experience Description
- Personalized landing based on user role.
- AI-driven case suggestions and community content.
- Clear calls-to-action for donations, help, and service discovery.
- Optimized for accessibility and search engine visibility.
- Seamless integration of social features and marketplace.

---

### UI System
- Consistent visual language across mobile and web.
- Design Tokens for colors, spacing, typography.
- Accessibility-first color contrasts.
- Responsive layout grid.
- Custom iconography aligned with Waggli branding.

### Screen-by-Screen Breakdown
- **Homepage**
  - Hero section with mission statement.
  - Search bar for cases & services.
  - Featured urgent cases.
  - Types of Help categories.
  - Heroes Nearby recognition.
  - Community highlights.
- **Case Pages**
  - Detailed pet information & situation description.
  - Photo gallery & medical documents.
  - Fundraising progress bar.
  - Action buttons: Donate, Share, Offer Help.
  - Comments & support messages.
- **Help Request Creation**
  - Role-specific flow for owners, shelters, etc.
  - Multi-step form with progress indicator.
  - Real-time form validation.
  - Review & publish preview.
- **Donation Flow**
  - Case summary display.
  - Preset & custom donation options.
  - Payment method selection (cards, SberPay, SBP).
  - Donor information with anonymity option.
  - Confirmation & success screen.
- **Blood Donor Pages**
  - Educational content on pet blood donation.
  - Blood donor registration form.
  - Eligibility & health checklist.
  - Location & availability settings.
- **Community & Social Areas**
  - Group directory & membership requests.
  - Group discussions & private messaging.
  - Community recognition & achievements.
- **Service Marketplace & Booking**
  - Search & filtering by location & service type.
  - Verified business profiles with reviews.
  - Service listings with availability.
  - Booking system for clinics, groomers, etc.
- **Admin Panel**
  - Role-based access control.
  - Full user, content, and case moderation.
  - AI moderation tool management.
  - Business/shelter verification workflows.
  - Escrow and financial oversight.
  - Platform-wide configuration & integrations.
- **Partner & Business Profiles**
  - Verified badges for legitimacy.
  - Reviews and ratings.
  - Service and promotional listings.
  - Direct contact options.
